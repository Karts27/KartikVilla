## Image Quality Tips

### Before Adding to Website:

1. **Compress Images** (Important for fast loading!)
   - Go to [TinyPNG.com](https://tinypng.com/)
   - Upload all your photos
   - Download compressed versions
   - Use compressed versions in your website

2. **Resize if Needed**
   - Hero images: Max 1920x1080px
   - Gallery images: Max 1200x800px
   - Use [Squoosh.app](https://squoosh.app/) for resizing

3. **Check File Names**
   - All lowercase: `hero.jpg` not `Hero.JPG`
   - No spaces: `gallery1.jpg` not `gallery 1.jpg`
   - Format: `.jpg` or `.png`

## Troubleshooting

### Can't Right-Click on Airbnb Images?
Try the DevTools method or screenshot method above.

### Images Too Large?
Compress at [TinyPNG.com](https://tinypng.com/) before adding to website.

### Image Not Showing on Website?
- Check file name matches exactly (case-sensitive)
- Make sure image is in `/images/` folder
- Refresh browser (Ctrl+R or Cmd+R)

### Want Better Photos?
Consider taking new professional photos if:
- Airbnb photos are low quality
- Listing is old
- Property has been updated
