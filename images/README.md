## Image Guidelines

### Recommended Sizes
- **Hero images**: 1920x1080px (landscape)
- **Gallery images**: 1200x800px (landscape)
- **Minimum**: 1000px width for best quality

### File Format
- Use `.jpg` for photos (smaller file size)
- Use `.png` for graphics with transparency
- Use `.webp` for best compression (modern browsers)

### Optimization
Before uploading, compress your images to reduce file size:
- [TinyPNG](https://tinypng.com/) - Free online compression
- [Squoosh](https://squoosh.app/) - Advanced compression tool
- Target: Keep images under 500KB each for fast loading

### Tips
- Use high-quality, well-lit photos
- Show different angles and rooms
- Include outdoor/exterior shots
- Capture unique features and amenities
- Professional photography recommended

## How to Add Images

1. Copy your photos to this folder
2. Rename them to match the names above (hero.jpg, gallery1.jpg, etc.)
3. That's it! The website will automatically display them

## Getting Images from Airbnb

If your images are currently on Airbnb:

1. Open your Airbnb listing
2. Right-click on each image
3. Select "Save image as..."
4. Save them to this folder with the correct names

OR use a browser extension to download all images at once:
- [Image Downloader](https://chrome.google.com/webstore/detail/image-downloader)
- [Download All Images](https://addons.mozilla.org/en-US/firefox/addon/save-all-images-webextension/)

## Missing Images?

If you don't have images yet:
- The website will still work (placeholder backgrounds)
- Add images later when you have them
- You can test with any images first, then replace them
